# V1.x Sub-Plan 6 — i18n Audit

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans. Implements umbrella candidate **#6**.

**Goal:** Sweep the frontend for any English string still rendered to the user (button labels, menu items, placeholder text, toast messages) and translate them to Chinese. Keep technical identifiers (variable names, console.log calls, comments) untouched.

**Architecture:** Pure `.vue` + `.ts` find-and-replace in `src/`. No new files. No new dependencies.

**Tech Stack:** `rg` for search.

---

## Task 1: Find candidates

- [ ] **Step 1: Grep for English UI strings**

Run: `rg -n "[A-Z][a-z]+(?: [A-Z][a-z]+){1,}" src/ --type vue --type typescript -g '!node_modules' -g '!*.d.ts'`
Expected output: a list of `path:line: snippet` for each match. Anything in `<template>` blocks, `message` args, button `>X<`, toast `content`, or placeholder text is a candidate.

- [ ] **Step 2: Classify each match**

For each match, decide:
- **Translate** (button labels, dialog titles, empty-state text, toast messages)
- **Keep English** (variable names, dev-only console messages, Naive UI component prop values that aren't user-facing)

Add a short comment next to "Keep English" entries in the plan todo list so reviewers can audit later.

---

## Task 2: Apply translations

**Files:** varies (all under `src/views/`, `src/components/`, `src/stores/`).

- [ ] **Step 1: Translate each "Translate" candidate**

Use idiomatic Chinese (Simplified). Examples already in the codebase:
- "Delete" → "删除"
- "Cancel" → "取消"
- "Confirm" → "确认"
- "Search" → "搜索"
- "Settings" → "设置"
- "Mark for delete" → "标记为删除"
- "Mark as viewed" → "标记为已看过"
- "Permanent delete" → "永久删除"
- "Restore" → "还原"
- "api_url" in Settings → "接口 URL" (literal: "interface URL")

- [ ] **Step 2: Re-run the grep**

Run the same `rg` command from Task 1 step 1.
Expected: zero new matches in user-facing strings. If any remain, iterate.

- [ ] **Step 3: Verify no Vue lint errors**

Run: `pnpm lint`
Expected: clean (or only pre-existing warnings).

- [ ] **Step 4: Commit**

```bash
git add src/
git commit -m "i18n: translate remaining English UI strings to Chinese"
```
